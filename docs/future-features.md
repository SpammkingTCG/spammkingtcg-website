# SpammKing TCG Future Features

## Rule for New Features

Only build a feature when it clearly supports one of these goals:

- Help customers buy cards.
- Help customers trust SpammKing TCG.
- Help suppliers trust SpammKing TCG.
- Improve maintainability or business operations.

## Near-Term Business Priorities

- Replace placeholder products with real products.
- Add real eBay listing URLs.
- Add real product photography.
- Confirm customer contact details.
- Submit the site to Google Search Console.
- Add Cloudflare Analytics or another analytics tool when ready.
